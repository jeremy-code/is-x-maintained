## Installation

```bash
git clone https://github.com/jeremy-code/is-x-maintained.git
cd is-x-maintained
corepack enable
yarn
```

## Usage

```bash
# Set Node.js version
nvm use

# Run the development server locally
yarn dev

# Lint and format code
yarn lint
yarn format

# Build
yarn build
```
